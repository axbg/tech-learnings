Installing Wordpress on Ubuntu Server 16.04



-> Create a database for wordpress
	-> open mysql with "mysql -u <<username>> -p"; here the username is root
CREATE DATABASE wordpress DEFAULT CHARACTER SET utf8 COLLATE utf8_unicode_ci;



-> Create an user for the wordpress process
GRANT ALL ON wordpress.* TO 'wordpress_user'@'localhost' IDENTIFIED BY 'wordpress';
FLUSH PRIVILEGES;



-> Update installed softwares and install wordpress php dependencies
sudo apt-get update
sudo apt-get install php-curl php-gd php-mbstring php-mcrypt php-xml php-xmlrpc
sudo service apache2 restart



-> Enable htaccess overrides
sudo vim /etc/apache2/apache.conf
	-> add the following lines at the bottom of the file
		<Directory /var/www/html/>
    		AllowOverride All
		</Directory>



-> Enable the rewrite module
sudo a2enmod rewrite



-> Test the new apache syntax and restart the server
sudo apache2ctl configtest
sudo service apache2 restart



-> Download and unzip wordpress 
cd /tmp
	-> use "curl -O https://wordpress.org/latest.tar.gz"
		** if curl doesn't work for whatever reason use
			-> "wget https://wordpress.org/latest.tar.gz"
tar xzvf latest.tar.gz



-> Use configurations for WordPress as mentioned in this tutorial starting with Step 4.
	https://www.digitalocean.com/community/tutorials/how-to-install-wordpress-with-lamp-on-ubuntu-16-04
		**instead of curl, use wget is curl doesn't have https connections enabled
